#### What I'm building

**ZinoFlow** is a full-featured helpdesk and ITSM platform for SMBs and enterprise teams. Multi-tenant SaaS with an on-premise option. Built with a small team with a strong focus on architecture, security and long-term maintainability.

**Core stack:** Next.js 15, TypeScript, PostgreSQL (Supabase), tRPC, Drizzle ORM, Stripe, Vercel

**Highlights:**
- Multi-channel support (Email, Live Chat, Widget, Portal, Slack, Teams, Shopify, REST API)
- AI-assisted ticket handling, KB generation, anomaly detection
- ITIL modules: Problem, Change, Asset Management, Automation Engine
- Full analytics suite with custom dashboards
- SAML SSO, MFA, GDPR compliance, Row-Level Security
- Embeddable chat widget (Preact, Shadow DOM, <24 KB gzipped)
